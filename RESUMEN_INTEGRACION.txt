1. 🔍 BÚSQUEDA POR VIN
   ┌─────────────────────────────────────────────────────────────┐
   │  • Usuario ingresa VIN de 17 caracteres                     │
   │  • Clic en botón "🔍 Buscar por VIN"                        │
   │  • Sistema consulta API de NHTSA                            │
   │  • Campos se llenan automáticamente:                        │
   │    ✓ Marca (ej: Toyota, Honda, Ford)                        │
   │    ✓ Modelo (ej: Camry, Civic, F-150)                       │
   │    ✓ Año (ej: 2020, 2021, 2022)                             │
   └─────────────────────────────────────────────────────────────┘

2. 📋 OBTENER MARCAS
   • Consulta todas las marcas disponibles en NHTSA
   • Más de 100 marcas (Honda, Toyota, Ford, BMW, etc.)
   • Uso potencial: autocompletado en formularios

3. 🚗 OBTENER MODELOS
   • Consulta modelos por marca y año
   • Ejemplo: Honda 2020 → 103 modelos
   • Uso potencial: validación de datos

🚀 CÓMO USAR LA NUEVA FUNCIONALIDAD:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  PASO 1: Ejecutar la aplicación
          mvnw.cmd javafx:run

  PASO 2: Ir a la sección "Vehículos"

  PASO 3: Clic en "Nuevo Vehículo"

  PASO 4: En el formulario verás:
          ┌─────────────────────────────────────────┐
          │ VIN: [________________] [🔍 Buscar VIN] │
          │ Placa: [______________]                 │
          │ Marca: [______________]                 │
          │ Modelo: [_____________]                 │
          │ Año: [___]                              │
          │ Cliente: [____________]                 │
          └─────────────────────────────────────────┘

  PASO 5: Ingresa un VIN (17 caracteres) y haz clic en "Buscar por VIN"

  PASO 6: ¡Los campos se llenan automáticamente! ✨

💡 CASOS DE USO REALES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  📌 Caso 1: Cliente nuevo llega al taller
     → Busca el VIN en el parabrisas del vehículo
     → Ingresa el VIN en el sistema
     → Sistema completa marca, modelo y año automáticamente
     → Ahorro de tiempo: 30-60 segundos por vehículo

  📌 Caso 2: Validar datos de vehículo existente
     → Cliente dice "Es un Honda Civic 2020"
     → Busca por VIN para confirmar
     → Verifica que los datos sean correctos

  📌 Caso 3: Cliente no sabe el modelo exacto
     → Cliente: "Es un Honda del 2020, no sé qué modelo"
     → Busca por VIN
     → Sistema identifica el modelo correcto
